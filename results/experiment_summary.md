# 阶段实验结果汇总

本文件记录目前已经完成的主要实验路线，便于中期检查和后续 PPT 汇报。

## 1. 官方 baseline 调参

| 实验方向 | 观察 |
| --- | --- |
| 增加 dataset1 epochs | 分数整体提升 |
| batch_size 200 vs 100 | batch_size=200 更稳定 |
| early_stop 放宽 | 有助于 dataset1 充分训练 |
| 多次同参数训练 | 存在随机波动，需要固定 seed 和保存 checkpoint |

早期 A 榜反馈从约 `1.042` 提升到约 `1.0878`。

## 2. 双模型融合

在官方 CRAFT 结果基础上，将历史最优结果与新训练 raw 结果做固定比例融合。

| 融合比例 | A 榜反馈 |
| --- | --- |
| 50% | 约 `1.12356` |
| 49% | 约 `1.12352` |
| 51% | 约 `1.12349` |

结论：固定比例双模型融合有效，但在 50% 附近趋于饱和。

## 3. 多样性模型融合

引入不同随机种子、邻居数量和训练轮数，构造差异性更强的 CRAFT 预测。

| 阶段 | A 榜反馈 |
| --- | --- |
| 5% 新模型融合 | 约 `1.12500` |
| 15% 新模型融合 | 约 `1.12631` |
| 21.5% 新模型融合 | 约 `1.12663` |

结论：分数随融合比例上升后出现峰值，超过峰值会下降，说明新模型有互补性但不能过量替换原最优结果。

## 4. 新种子模型与自适应融合

| 实验 | 方法 | A 榜反馈 |
| --- | --- | --- |
| round17 | best21.5% + seed7777 低比例融合 | `1.1271744963034764` |
| round18 | rank-space 融合 | `1.124651558251298` |
| round19 | 置信度自适应融合 | `1.1273681134893112` |
| round20 | 提高 extra 权重 | `1.1273511801941303` |

结论：rank-space 融合不适合当前结果；置信度自适应融合优于固定比例融合；继续盲目提高权重会下降。

## 5. 当前最好方案

```text
round19_adapt_best215_seed7777_b02_e06_t005
score = 1.1273681134893112
```

方法解释：

1. 保留此前最优的全局融合结果。
2. 引入 seed7777 训练得到的 CRAFT 模型作为互补模型。
3. 对不同测试样本，根据候选分数差距和置信度动态调整融合强度。
4. 对置信度较高的样本保守融合，对不确定样本增加新模型贡献。

## 6. 下一步实验

1. 不再大范围测试 rank-space 融合。
2. 围绕 round19 的自适应参数做小范围搜索。
3. 继续训练一个结构差异更大的 CRAFT 配置，而不是只微调融合比例。
4. 将本地验证 MRR 与 A 榜反馈同步记录，避免只凭提交结果判断。
