# 中期汇报材料

## 研究题目

基于图学习的动态推荐任务研究：面向时序图未来链接预测的时间感知候选节点重排序策略。

## 任务定义

真实推荐系统、社交网络、论文引用网络和网页跳转网络中的交互关系会随时间持续演化。项目将历史交互表示为 `(source, destination, time)` 三元组，在给定源节点、预测时间和候选目标节点集合时，预测未来最可能发生交互的目标节点顺序。

比赛核心指标为 MRR，辅助指标包括 Recall@K、NDCG@K、HitRate@K、训练时长、推理时间和显存占用。

## 技术路线

```text
原始时序交互三元组
-> 字段检查与清洗
-> ID 映射与时间排序
-> 时间切分训练/验证/测试
-> 官方 CRAFT baseline
-> 历史频次、近期频次、时间衰减等基础排序
-> 多随机种子 CRAFT 训练
-> 候选节点概率打分
-> 固定权重融合与置信度自适应融合
-> MRR/Recall@K/NDCG@K/HitRate@K 评测
-> 消融分析与提交文件生成
```

## 阶段实验结果

| 阶段 | 方法 | 结果 |
| --- | --- | --- |
| 官方 baseline 复现 | 调整 epochs、early_stop、batch_size | A 榜约 1.0878 |
| 双模型固定融合 | 历史最优与新 raw 结果融合 | A 榜约 1.12356 |
| 多样性模型融合 | 加入不同随机种子 CRAFT 预测 | A 榜约 1.12663 |
| 新种子低比例融合 | best21.5% + seed7777 | A 榜 1.1271744963034764 |
| 置信度自适应融合 | 根据样本置信度调整融合比例 | A 榜 1.1273681134893112 |

## 阶段性观察

1. 官方 baseline 本身已经较强，单纯延长训练轮数的提升有限。
2. 时序图任务中近期行为和重复交互对排序结果有明显影响。
3. 不同随机种子模型输出存在互补性，多模型融合明显优于单模型。
4. 固定比例融合存在峰值，超过峰值后分数下降。
5. 置信度自适应融合能针对不同样本使用不同融合强度，是当前最有效的策略。
