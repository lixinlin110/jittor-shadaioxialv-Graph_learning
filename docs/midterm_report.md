# 中期汇报材料

## 研究题目

基于图学习的动态推荐任务研究：面向时序图未来链接预测的时间感知候选节点重排序策略。

## 与开题报告的一致性

本仓库内容与《开题报告.docx》中的研究方向一致。项目不是多模态 RAG，而是围绕 Jittor/JittorGeometric、动态图推荐、时序图未来链接预测和候选节点重排序展开。详细逐项对照见 [opening_report_alignment.md](opening_report_alignment.md)。

## 任务定义

历史交互表示为 `(source, destination, time)` 三元组。测试阶段给定源节点、预测时间和 100 个候选目标节点，模型需要输出每个候选 destination 与 source 在该时间发生交互的概率，并根据概率完成候选节点重排序。

比赛核心指标为 MRR，辅助指标包括 Recall@K、NDCG@K、HitRate@K、AUC/AP、训练时间、推理时间、显存占用、重复交互命中率和新交互命中率等。

## 技术路线

```text
原始时序交互三元组
-> 字段检查与清洗
-> ID 映射与时间排序
-> 按时间切分训练/验证/测试
-> 候选目标节点集合检查
-> 历史频次、近期频次、时间衰减等基础排序
-> BPR/矩阵分解轻量对照、LightGCN 后续计划对照
-> 官方 CRAFT/JittorGeometric baseline
-> 多随机种子 CRAFT 训练
-> 候选节点概率打分
-> 固定权重融合与置信度自适应融合
-> MRR/Recall@K/NDCG@K/HitRate@K/AUC 评测
-> 消融分析与提交文件生成
```

## 阶段实验结果

| 阶段 | 方法 | 结果 |
| --- | --- | --- |
| 官方 baseline 复现 | 调整 epochs、early_stop、batch_size | A 榜约 1.0878 |
| 双模型固定融合 | 历史最优与新 raw 结果融合 | A 榜约 1.12356 |
| 多样性模型融合 | 加入不同随机种子 CRAFT 预测 | A 榜约 1.12663 |
| 新种子低比例融合 | best21.5% + seed7777 | A 榜 1.1271744963034764 |
| 置信度自适应融合 | 根据样本置信度调整融合比例 | A 榜 1.1273681134893112 |

## 阶段性观察

1. 官方 CRAFT baseline 本身较强，单纯延长训练轮数提升有限。
2. 时序图任务中近期行为和重复交互对排序结果有明显影响。
3. 不同随机种子模型输出存在互补性，多模型融合优于单模型。
4. 固定比例融合存在峰值，超过峰值后分数会下降。
5. 置信度自适应融合能针对不同样本使用不同融合强度，是当前最有效的策略。
