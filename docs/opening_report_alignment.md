# 开题报告一致性对照

本文档用于把仓库当前内容与《开题报告.docx》逐项对照，方便中期检查、老师复核和后续答辩说明。

## 总体结论

仓库当前方向与开题报告主线一致：研究对象均为“基于图学习的动态推荐任务”，核心任务均为时序图未来链接预测和 100 个候选 destination 的概率打分/重排序。当前代码以 Jittor/JittorGeometric CRAFT baseline 为主体，并围绕历史频次、近期频次、时间衰减、多随机种子模型集成和置信度自适应融合开展实验。

需要特别说明的是，开题报告中写到的 BPR/矩阵分解、LightGCN 属于计划中的对照基线；当前阶段已经优先完成官方 CRAFT、频次基线、时间感知重排序、模型集成和实验记录，后续会补充 BPR/LightGCN 的轻量对照实验。

## 需要避免的表述

1. 不把当前项目描述为“多模态 RAG”，本仓库主题是动态图推荐和时序图链接预测。
2. 不把 A 榜结果融合描述为“直接改答案”，而应描述为“多模型预测结果集成、候选节点重排序和消融实验”。
3. 不声称 BPR/LightGCN 已经完整实现；当前应写为“计划中的对照基线/后续补充实验”。
4. 不随机打乱官方数据；数据已经按时间顺序排列，本地验证也必须按时间切分。
